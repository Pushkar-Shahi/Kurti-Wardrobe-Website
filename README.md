# Kurti Store Template

A beautiful, mobile-friendly, digital catalogue and storefront template for selling Kurtis and boutique clothing. 

## Structure
- `index.html`: Contains the layout and structure of the application.
- `styles.css`: Contains the styling and responsive design rules.
- `script.js`: Contains the product data and UI interaction logic.

## How to Customize

### 1. Changing the Phone Number for WhatsApp Orders
Currently, the template uses a placeholder phone number (`1234567890`) for WhatsApp integration. 

To change this to your actual phone number so you can receive orders:
1. Open the `script.js` file.
2. Locate the configuration variable for the phone number (usually near the top).
   ```javascript
   const YOUR_PHONE_NUMBER = "1234567890";
